Instructions:
-If Opened From Console-
1. Menu will be printed to console with all avalible options.
2. Select calculation mode (infix is deafult mode).
3. Set expression.
4. Calculate expression with option "4".
5. Repeat steps 2-4 until you wish to exit, exit with option "5"
Note: to have menu reprint input anything that isnt 1-5

-If Opened Anywhere Else-
1. Enter expression into text box.
2. Select mode using radio buttons
3. Click calculate button and recive result of your expression
4. Repeat steps 1-3 until you wish to close the application
